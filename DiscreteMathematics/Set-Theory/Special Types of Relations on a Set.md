# Reflexive and Ir-reflexive Relation

> [!lecture] Lecture-21
## Reflexive Relation
- A relation $R$ on a set $A$ is said to be reflexive iff all the elements of $A$ is related to itself.
$$
\forall x \in A\; (x \; R\; x)
$$
